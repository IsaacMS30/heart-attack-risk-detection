# Heart Attack Risk Detection

This project aims to build several machine learning model to predict whether a patient has a heart disease or not. These predictions are based on their medical information.

To achieve this, we build 3 different models:
* KNN
* Random Forest
* Logistic Regression

After training the models, their metrics are evaluated to determine which one presented the best performance.

## Data

The data used in this project can be found here: https://www.kaggle.com/datasets/jocelyndumlao/cardiovascular-disease-dataset/data
